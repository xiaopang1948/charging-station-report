## 七、针对四子王旗的购买建议

### 7.1 120kW vs 160kW 决策建议

| 因素 | 建议 |
|------|------|
| **功率需求** | 四子王旗目前电车保有量不明，120kW双枪完全足够（同时充2辆车） |
| **价格因素** | 160kW价格是120kW的2.5倍，性价比低 |
| **电力条件** | 160kW需要更大容量，需先确认商铺供电 |
| **充电速度** | 160kW比120kW快约25%，但对县城用户来说差异不大 |
| **最终建议** | **优先考虑120kW双枪**（¥27,500招标价），除非电力条件特别适合160kW |

### 7.2 购买步骤

1. **打400-01-95598** 确认：
   - 单台120kW或160kW零售能不能买
   - 通过什么渠道（1688/经销商/直接下单）
   - 内蒙古乌兰察布有没有代理商或售后点

2. **如果许继零售买不到**：
   - 备选方案①：通过1688找许继官方店询价
   - 备选方案②：改选鸿嘉利（次选推荐）
   - 备选方案③：联系许继在内蒙古的项目合作伙伴（蒙电e充的供应商渠道）

3. **确认价格**：
   - 120kW双枪：招标价¥27,500，零售价预估¥30,000~35,000
   - 160kW双枪：招标价¥69,380，零售价预估¥70,000~80,000
   - 含运费安装：需另外加价

4. **下单前确认售后**：
   - 质保年限
   - 内蒙本地维修点
   - 响应时间承诺
